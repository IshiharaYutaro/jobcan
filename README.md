# jobcan

打刻ツール

## Usage

```
  -code string
        ログインのURLに含まれるcode値
  -date string
        打刻日時 format: 2006-01-02, default: 2018-06-21
  -gid string
        グループID
  -lat string
        緯度
  -lon string
        経度
  -reason string
        理由
  -taikin
        退勤フラグ default: false
  -time string
        打刻時間 format: 15:04, default: 12:21
  -yakin
        夜勤モード default: false
```
